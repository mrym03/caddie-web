# Caddie marketing site — handoff

**Project dir:** `~/Desktop/wesbite-1` (the folder is named "wesbite-1" — a typo, but it IS the Caddie site).
**What it is:** a single-page, static marketing site for **Caddie** — an AI voice copilot that joins live sales calls. No framework, no build step. Plain `index.html` + a few JS files.

## Run it locally
```bash
cd ~/Desktop/wesbite-1
./serve.sh            # python3 http.server on port 8765
# open http://localhost:8765
```

## Deploy
- Git remote `origin` → **https://github.com/mrym03/caddie-web** (branch `main`).
- **Netlify auto-deploys** from `main` (`netlify.toml` at root, publish = ".", no build).
- Push to deploy: `git add -A && git commit -m "..." && git push origin main`.

## File map
- `index.html` — the entire page: inline `<style>` (design tokens + all section CSS) and all section markup, in order: nav → hero → ticker → The Problem → What It Does (01–03) → How It Works (01–03) → See It In Action (call mockup) → The Dashboard (3 cards) → Pricing → "Meet Caddie" brand moment → Final CTA → Footer → SEO/OG meta.
- `js/three.min.js` — vendored Three.js r128 (don't edit).
- `js/gradient.js` — the **hero/CTA background**: a Three.js sphere with a simplex-noise gradient shader (green/orange/black, lifted verbatim from monopo-saigon) + a **refractive glass orb** (Lens3D, cubemap refraction + fresnel) + a full-screen **film-grain** post-pass. **No intro motion** — it renders the settled scene immediately. Reacts to cursor (parallax).
- `js/site.js` — nav blur-on-scroll, mobile menu, the ticker content, scroll-reveal (IntersectionObserver → `.in-view`), and pausing the gradient when the hero/CTA are off-screen (`window.GRADIENT_ACTIVE`).
- `js/liquid.js` — **liquid-glass (chrome) shader**, ported from `~/Desktop/web-visuals/liquid-logo` (paper.design). Renders the Caddie monogram as flowing liquid metal in the "Meet Caddie" section. Reads a precomputed height-field PNG; lazy-starts when scrolled into view.
- `img/logo-mark.svg` — white monogram only (nav/footer), transparent bg.
- `img/logo-full.svg` — full lockup (mark + "CADDIE"), transparent bg.
- `img/logo-mark-field.png` — precomputed bevel height-field of the monogram (input for `js/liquid.js`; generated by a Poisson solve so the browser doesn't have to).
- `favicon.*`, `apple-touch-icon.png`, `safari-pinned-tab.svg`, `site.webmanifest` — see gotcha below.

## Design system (IMPORTANT — these are the owner's explicit preferences)
- **Monochrome black & white. NO amber.** The original spec modeled it on Foyer (dark + amber `#f59e0b`); the owner explicitly rejected that ("we don't want it to be that much like Foyer"). Tokens: `--bg #0a0a0a`, `--cream #f4f4f4`, accents are white. Don't reintroduce amber.
- **Keep the green/orange gradient + glass orb hero** — the owner loves it; it's the one bit of colour. Everything else is B&W.
- **Font = serif** (Georgia/Times stack) for display + body — the owner wanted "the original font," NOT the spec's Inter. Mono only for tiny bracketed eyebrows/labels. Set via `--font-display` / `--font-body` / `--font-mono`.
- Hero headline is **italic**: "Notetaker that speaks and sells." (`.hero-title`, italic, `clamp(2.74rem, 7.27vw, 7.27rem)`).
- Logo = white rounded monogram (source: `~/Downloads/Black and White Minimalist Brand Logo/caddie.svg`).
- Primary CTAs link to the app: **https://app.heycaddie.live/dashboard** ("Talk to us" stays on the `#get-started` anchor).

## Gotchas / tips
- For visual QA I drove a headless browser via Playwright from the npx cache: `/Users/maryam/.npm/_npx/e41f203b7505f1fb/node_modules/playwright` (import the CJS default, then `const {chromium}=pkg`). Launch with `{channel:'chrome'}` for WebGL. The page references dead CDNs from its monopo origin? No — it's fully self-contained now (Three.js is vendored). 
- The gradient/orb is WebGL; if a section needs to sit on solid colour, give it an opaque background (the content sections use `var(--bg)`; only hero + CTA are transparent over the fixed gradient canvas).
- Shaders are extracted verbatim from monopo-saigon (gradient/orb) and paper.design liquid-logo (liquid chrome) — see `~/Desktop/web-visuals/` for the originals.
